# ArchaeoSchild – Frontend Practice Project

This repository contains a static company website created as a frontend practice project using HTML, CSS, JavaScript, and jQuery.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Features

- responsive navigation
- image gallery
- structured content sections
- references table
- expandable application form
- contact section with social links

## Project Goal

The purpose of this project was to practice frontend development basics, especially page structure, styling, navigation, and simple interactive elements.

## Note

This project was created for learning purposes, so some content elements are placeholders.
